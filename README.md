## Quick Start

> [!note]
> Before running this shell script, make sure you have **zsh** and **oh-my-zsh** installed!

```bash
git clone https://github.com/islgl/configure-ohmyzsh.git
cd configure-ohmyzsh

chmod +x configuration.sh

./configuration.sh

source ~/.zshrc
```
